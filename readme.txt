Modern restaurant WordPress theme with menu, reservations, and responsive design.

== Description ==
Obydullah Restaurant is a modern, elegant WordPress theme designed for restaurants, cafés, and food businesses. It features a clean layout, responsive design, and customizable sections including menu, testimonials, chef specials, and contact information.

== Features ==

* Fully responsive design
* Customizer options for contact info
* Menu and testimonial sections
* Hero slider and event sections
* SEO-friendly structure
* Lightweight and fast loading
* Custom navigation menus

== Installation ==

1. Upload the theme folder to `/wp-content/themes/`
2. Activate the theme via Appearance → Themes
3. Install recommended plugins if prompted
4. Customize via Appearance → Customize

== Recommended Plugins ==

* Contact Form 7
* Obydullah Restaurant Core (if available)

== Customization ==
You can customize the theme via:
Appearance → Customize → Contact Info
